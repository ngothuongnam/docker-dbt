Welcome to your new dbt project!

### Using the starter project

Try running the following commands:
```
cd project/transform-stg-sv
export DBT_PROFILES_DIR=$(pwd)
dbt snapshot
dbt run
dbt test
dbt docs generate
dbt docs serve
dbt clean
dbt compile
```
